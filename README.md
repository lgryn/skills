# Skills Library

Personal repository for agent skills.

## Structure

```text
skills/
  nodejs/
    <skill-slug>/
      SKILL.md

templates/
  category/
    README.md
  skill/
    SKILL.md
```

## Skill Shape

Each `SKILL.md` should stay simple and usually follow:

- `Purpose`
- `Trigger`
- `Inputs`
- `Workflow`
- `Output`
- `Guardrails`
