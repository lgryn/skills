# Node.js Skills

This category is for skills related to Node.js, backend tooling, server applications, packages, CLI utilities, and Node runtime behavior.

## When To Use

- when a task is related to Node.js or the npm ecosystem;
- when environment setup, scripts, packages, or service structure need work;
- when a repeatable workflow is needed for Node.js projects.

## Current State

For now, this category acts as a base container for future skills.
New skills should be added in dedicated directories using this pattern:

```text
skills/nodejs/<skill-slug>/SKILL.md
```
